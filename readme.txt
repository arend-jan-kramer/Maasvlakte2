# Uitbreiding maasvlakte 2
De maasvlakte wordt steeds groter. Dit betekent dat alles wat binnen komt ook verder op transport moet.
De opdracht van school was om een web applicatie te bouwen waarop weergegven wordt hoe de treinen reiden en hoelaat zij op welk spoor aan komen.

# Code
HTML, CSS, Javascript/Jquery, PHP, MySql
